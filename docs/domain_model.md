# ドメインモデル

- タスク
  - 概要: やらないといけないことを表す
  - フィールド
    - ID
    - 概要
    - サブタスク
    - ステータス:
      - ステータスの制御を行う
      - created, todo, doing, done, closed
    - ラベル:
      - 諸々の機能は、ラベルで対応する
- サブタスク
  - 概要: タスクに紐ずくサブのタスク
  - フィールド
    - ID
    - 概要: markdown を入れる
- タスクプロバイダー
  - 概要: タスクの提供元、ユーザ入力、Github、Jira など
- タスクステータス履歴
  - 概要: タスクのステータスが変わったときに発行される
